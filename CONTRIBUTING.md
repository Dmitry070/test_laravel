# Рекомендации по внесению вклада

Спасибо за интерес к нашему проекту! Вот несколько рекомендаций для внесения вклада:

## Как начать

1. **Fork проект** на GitHub
2. **Клонируйте** ваш fork локально:
   ```bash
   git clone https://github.com/ваше-имя/example-app.git
   cd example-app
   ```

3. **Установите зависимости**:
   ```bash
   composer install
   npm install
   ```

4. **Скопируйте `.env.example` в `.env`**:
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

5. **Создайте ветку** для вашей функции:
   ```bash
   git checkout -b feature/вашя-фича
   ```

## Процесс разработки

1. Убедитесь, что ваш код следует стилю проекта
2. Добавьте комментарии на русском языке
3. Протестируйте ваши изменения
4. Убедитесь, что все тесты проходят

## Commit сообщения

Используйте понятные commit сообщения на русском:
```bash
git commit -m "Добавляю функцию X"
git commit -m "Исправляю ошибку в Y"
git commit -m "Улучшаю производительность Z"
```

## Pull Request

1. **Push** вашу ветку на fork:
   ```bash
   git push origin feature/вашя-фича
   ```

2. **Откройте Pull Request** с описанием изменений

3. **Ждите review** и отвечайте на комментарии

## Код стиля

- ✅ PSR-12 для PHP кода
- ✅ Комментарии на русском языке
- ✅ Максимум 120 символов на строку
- ✅ Используйте типизацию (type hints)

## Тестирование

```bash
# Запустить тесты
php artisan test

# С coverage
php artisan test --coverage
```
